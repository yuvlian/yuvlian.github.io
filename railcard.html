Railcard

Railcard is a Honkai: Star Rail character showcase cards maker. Written in Python and powered by mihomo. Yes it supports custom images. Yes it indicates relic rarity with the background.

Examples:


To get started, run terminal as admin and type these in:

  • git clone https://github.com/yuvlian/railcard.git
  • cd railcard
  • pip install -r requirements.txt
  • python main.py
