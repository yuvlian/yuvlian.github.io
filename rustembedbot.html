Rusty Embed Fixer Bot

Rusty Embed Fixer Bot is a very simple Discord bot written in Rust that resends your message that contains tiktok/instagram/twitter/x link(s) with the embed fixed by replacing the links with tnktok, ddinstagram, vxtwitter, and fixvx.

How To Use

Prebuilt

  • Download this
  • Extract the zip
  • Edit config.yml to your actual discord bot token
  • Launch the binary (aka .exe)
    • (Yes I know it's named "rusty_tiktok_bot" despite supporting 2 other websites... too lazy to change it.)
  • Bot is online!

Build From Source

  • git clone https://github.com/yuvlian/rusty_embed_fixer_bot.git
  • You already know the rest if you're building from source
